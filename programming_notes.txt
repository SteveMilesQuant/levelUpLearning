

Create a project:
	python3 -m virtualenv virt
	source virt/bin/activate
	python3 -m pip install fastapi
	python3 -m pip install "uvicorn[standard]"
	python3 -m pip install jinja2
	python3 -m pip install oauthlib
	python3 -m pip install aiohttp
	python3 -m pip install python-multipart
	python3 -m pip install pandas
	python3 -m pip freeze > requirements.txt

Working directory:
	cd /mnt/c/Users/Steve/teacherCamp/
	source virt/bin/activate

Launch using uvicorn:
	uvicorn main:app --reload --port 8080
	uvicorn main:app --reload --port 8080 --ssl-keyfile=./megducation.com+5-key.pem --ssl-certfile=./megducation.com+5.pem

Webpage:
	https://localhost:8080

Hacky way of getting public interface for OAuth2 credentials:
	Edit C:\Windows\System32\drivers\etc\hosts with:
